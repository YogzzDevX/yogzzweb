Get Started

Ubah Startup mu menjadi node run.js untuk bisa mengakses bash

File Bash run.js

              
              const { spawn } = require('child_process')
const cp = require('child_process')
const { promisify } = require('util')
const exec = promisify(cp.exec).bind(cp)

function start(cmd) {
	return spawn(cmd, [], {
		stdio: ['inherit', 'inherit', 'inherit', 'ipc']
	})
}

start('clear')
start('bash')
console.log('Terminal ready to use!')

Tombol Antarmuka

Terdapat Tiga button / tombol dengan fungsi berbeda

Options Fungsi lain Otomatis
Start Berfungsi untuk memulai Server. false
Restart Berfungsi untuk me-restart / memulai kembali server panel ketika di butuhkan. false
Stop Berfungsi sebagai tombol darurat ( kill, stop ) dimana tombol tersebut untuk menghentikan server yg sedang berjalan. true

Setiap button memiliki fungsi berbeda sesuai penamaan nya.

Keybindings

Keybindings biasa digunakan untuk user yang menggunakan Komputer, laptop, MacBook dll.

Berfungsi sebagai shortcut dan mempermudah dalam multitasking.
